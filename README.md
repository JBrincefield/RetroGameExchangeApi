# Retro Video Game Exchange API

## Description

A RESTful API built with ASP.NET Core for managing users and retro video games available for trade.

This project was developed for CSC380 and follows REST principles including
Richardson Maturity Model Level 3 (HATEOAS).

## Features

- User registration and profile updates
- Create, update, delete retro video game listings
- Search for video games
- JSON-based API
- OpenAPI documentation via Swagger

## Technology Stack

- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQLite
- Swagger / OpenAPI

## Running the Project

1. Open the solution in Visual Studio
2. Run the project
3. Navigate to `/swagger` to view API documentation
